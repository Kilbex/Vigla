# Project operations

These runbooks keep launch and maintenance work reproducible without adding
product telemetry or a binary-distribution channel.

- [Creator and reviewer kit](./creator-kit.md) — a truthful 10-minute demo,
  credential-free recording paths, public evidence, and media inventory.
- [GitHub launch controls](./github-launch.md) — owner settings, branch rules,
  Discussions, and public launch checks.
- [GitHub traffic snapshots](./github-traffic.md) — privacy-preserving repository
  traffic archiving in a separate private repository.

Repository operations never upload a Vigla application artifact. Users build
the macOS disk image locally from the reviewed source.
