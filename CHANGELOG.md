All notable changes to Vigla are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

Initial public release candidate.

- Mission supervision: supervisor-driven decomposition, arbiter-bounded
  Accept / Extend / Scrub / Escalate, structured completion verdicts.
- Cross-vendor workers: Claude Code, Codex CLI, and end-to-end-verified
  Antigravity, plus legacy Gemini CLI support, profile-backed Kiro / Copilot,
  and a deterministic mock harness for credential-free demos.
- Local-only macOS packaging: one-command, ad-hoc-signed DMG builds with no
  maintainer release artifact or personal signing identity.
- Daily GitHub traffic snapshots to a private metrics archive, with overlapping
  14-day windows deduplicated in generated reports.
- Reversibility: worktree-per-worker isolation, pre-merge snapshot
  tags, one-click mission revert.
- Memory Kernel: local event-sourced repository memory with witness
  scoring, promotion policy, and vendor-native file rendering.
- Operations Room UI: station canvas, worker drawer, review queue,
  mission inbox, history, and replay.
- Plan review workspace: a wider, accessible execution map, four-bound envelope
  readout, progressive plan details, and clearer approval controls.
- Defensive plan projection: sanitized model-authored labels and bounded graph
  size before layout.
