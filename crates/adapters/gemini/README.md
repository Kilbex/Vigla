# Gemini CLI adapter

Maintained compatibility adapter for Gemini CLI stream JSON. Consumer
Login-with-Google access ended on 2026-06-18, so this is a legacy/enterprise
lane rather than Vigla's current Google-path integration.

```sh
cargo test -p vigla-adapter-gemini
```

Keep existing fixtures and forward-compatible parsing healthy; do not present
this adapter as a current consumer setup path.
